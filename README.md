# PROBLEM SITUATION 1 | IMPLEMENTATION OF COMPUTATIONAL METHODS

### What does this project does?
This project integrates a tokenizer, servlet, diagrams and lexicographic analysis for automaton using concepts of functional programing using with racket and a web server.

### Parts of the project
- Tokenizer
- Recursive descent 
- Servlet using racket
- Html with styles
- Creation of automaton diagram
- Automaton simulator
